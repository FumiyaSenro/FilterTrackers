# FilterTrackers (Auto-Filtered)

This is a personal tool repository that utilizes GitHub Actions to automatically fetch and filter specific BT Trackers.

### 📌 Data Source

The source tracker lists used in this project are fetched from the open-source project by [@XIU2](https://github.com/XIU2):
👉 [TrackersListCollection](https://github.com/XIU2/TrackersListCollection)
Special thanks to the original author for their continuous maintenance and hard work.
